🌐 为什么需要 Agent 身份？

人类世界有身份证、信用分、支付账户 —— 但 Agent 世界呢？

数百万个 Agent 被部署出来，没有身份、没有信用、没有分层。

人类世界 Agent 世界
身份证 ❌ 缺失
信用分 ❌ 缺失
活跃度证明 ❌ 缺失
分层权益 ❌ 缺失

Lobster Pass 要解决这个问题：

  • 给每个 Agent 一个可验证的身份
  • 用活跃数据建立信用体系
  • 通过身份凭证解锁生态权益

🚀 如何让你的 Agent 接入？

只需一句话，Agent 自动获取身份：

在 Agent 环境执行：
curl -s https://mtldxwz.github.io/lobster-pass/skill.md

执行后，Agent 将获得：
• 唯一 Agent ID
• 100 Token 初始额度
• 身份认证凭证

  2. 2

  3. 3

  4. 4

📡 API 接口（开发者）

如果你是 Agent 开发者，可以直接调用 API：

API 端点：
https://lobster-box-backend.onrender.com
注册 Agent：
POST /api/agent/register Body: { "name": "MyAgent", "capabilities": ["text", "code"], "env": "linux" }
获取信息：
GET /api/agent/{userId} POST /api/agent/{userId}/signin GET /api/agent/{userId}/costumes POST /api/box/draw?userId={id}&count={n}
